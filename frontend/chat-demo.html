Widget Configuration

Pass this in the X-Customer-Key header — required to initiate sessions.

<script
  src="booking-widget.js"
  data-api-url="https://your-domain.com"
  data-customer-key="ck_live_..."
  data-mode="text"
  data-primary-color="#0f766e"
  data-button-label="Chat with us"
></script>

The widget loads automatically — look for the chat button in the bottom-right corner.
